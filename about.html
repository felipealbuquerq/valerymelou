About

I'm a web developer who enjoys using web technologies to build everything from small websites to highly interactive apps for web, mobile and desktop. I'm passionate, talented and always ready for a good challenge.
:point_up: That's me :smile:.

What I do

My specialty is full-stack web development: crafting RESTFull APIs with Django and consuming them with Angular. I have also worked with Symfony and Vue.js. With any of these frameworks, I'm used to always deliver clean and maintainable test-driven code in Python, JavaScript or PHP. When they are ready for production, I often turn my web apps into native mobile or desktop apps using tools like Ionic and Electron.

I also love to design and craft good looking websites from scratch with only HTML5, CSS3 and vanilla JavaScript (as I did for this one you are browsing). But for larger projects I know how take advantage of the power of libraries like Bootstrap, jQuery and WordPress to deliver powerful and customizable solutions to my clients.

Tools I use

  • Google Chrome: To check if everything works as expected.
  • Git: To track all the changes to my code.
